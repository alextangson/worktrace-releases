WorkTrace 支持

下载、安装、权限与问题反馈

下载与更新

请从 GitHub Releases 获取公开版本和校验文件。Windows 版本也可通过 Microsoft Store 产品页 获取（以商店审核与发布状态为准）。

macOS 安装

  1. 打开 DMG，将 WorkTrace 拖入“应用程序”。
  2. 首次运行时按引导授予辅助功能、屏幕录制等你希望启用的权限。
  3. 升级时先退出 WorkTrace，再用新版本覆盖“应用程序”中的旧版本；本地数据与系统授权通常会保留。

反馈问题

请到 GitHub Issues 提交问题，并提供操作系统版本、WorkTrace 版本和复现步骤。不要公开上传 API Key、客户资料、窗口标题、浏览器地址或包含敏感内容的截图。

隐私与数据

你可以在 WorkTrace 中暂停采集、排除敏感应用、关闭屏幕快照或清除本地数据。完整说明请阅读 WorkTrace 隐私政策。
