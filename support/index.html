S-tello Trace 支持

下载、安装、更新、权限与问题反馈

下载与更新

请从 GitHub Releases 获取公开版本和校验文件。S-tello Trace 会定期检查签名的新版本；发现更新时，可按应用内提示下载并安装。如果应用内更新不可用，可回到 Releases 手动下载。Windows 版本将通过 Microsoft Store 产品页 提供（以商店审核与发布状态为准）。

macOS 安装

  1. 打开 DMG，将 S-tello Trace 拖入“应用程序”。
  2. 首次运行时按引导授予辅助功能、屏幕录制等你希望启用的权限。
  3. 应用内更新会自动完成替换。手动升级时，先退出 S-tello Trace，再用新版本覆盖“应用程序”中的旧版本；本地数据通常会保留。

反馈问题

请到 GitHub Issues 提交问题，并提供操作系统版本、S-tello Trace 版本和复现步骤。不要公开上传 API Key、客户资料、窗口标题、浏览器地址或包含敏感内容的截图。

隐私与数据

你可以在 S-tello Trace 中暂停采集、排除敏感应用、关闭屏幕快照或清除本地数据。完整说明请阅读 S-tello Trace 隐私政策。
